# Student App Web

### Description

It is a react app where you can see all students with their first name, last name, birth
date, hobbies, photo. Also you can add new student, edit or delete a student.

### Technology Stack

-   React.js
-   Redux
-   Sass
-   Flow
-   Eslint
-   Jest
